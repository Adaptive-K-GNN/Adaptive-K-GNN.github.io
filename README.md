# Adaptive-K Project Page

Official project page for:

**When Task Performance Deceives: Task-Geometry Decoupling in Learnable-Curvature Hyperbolic GNNs**  
*Neural Networks*, 203 (2026), Article 109172.

## Deploy with GitHub Pages

This package is prepared for the repository:

`Adaptive-K-GNN/Adaptive-K-GNN`

Upload the files and folders in this package to the repository root, replacing the existing `index.html` when prompted. Then enable GitHub Pages from the `master` branch and `/ (root)` folder.

Expected page URL:

`https://adaptive-k-gnn.github.io/Adaptive-K-GNN/`

## Included figures

- `task_geometry_decoupling.png`: the paper's task-geometry decoupling teaser.
- `adaptive_k_framework.png`: the Adaptive-K controller architecture.

The page intentionally uses only these two core figures.
